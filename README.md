# Marcin-Lechowski
Serwis rowerowy

## Website

Projekt strony znajduje się w katalogu `website`.

Quick commands:

```bash
cd website
npm install
npm run dev

# pobrać przykładowe zoptymalizowane obrazy
npm run fetch-images

# lub: wgrać własne oryginały do public/images/originals/ i skonwertować
npm run convert-images
```

Podczas deployu na Vercel ustaw root projektu na `website`.
